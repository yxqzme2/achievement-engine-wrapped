Instructions

// Setup Instructions //

  1. Download both .env and docker-compose.override.yml using the buttons above.
  2. Place both files in the same directory as the project's docker-compose.yml.
  3. Start the stack: docker compose up -d
  4. Open http://your-server-ip:8000 in your browser to reach the Achievement Engine landing page.
  5. The engine polls abs-stats every 300s — navigate to /journal after the first poll completes to see your data.
Unraid users: Use the Template Forge page to generate and download both Unraid templates (my-achievement-engine.xml and my-abs-stats.xml). Import them in Unraid's Community Applications to configure both containers through the Unraid UI instead of docker compose.
