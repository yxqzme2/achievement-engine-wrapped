Portainer Stack Generator

Achievement Engine + ABS Stats

This builder outputs a single Portainer-compatible stack file for both containers and an optional user_xp_start.json sidecar. It preserves the options exposed by the current Unraid templates: images, container names, network, ports, paths, SMTP, wrapped settings, user filters, ABS tokens, and Discord.
