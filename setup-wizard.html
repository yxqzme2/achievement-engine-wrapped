After Downloading Both Files

    


      

        
1

        

          
Move the downloaded files into your project folder (where you cloned the repo)

          
# Both files go here:
achievement-engine-wrapped/
  ├── .env                             ← downloaded
  ├── docker-compose.override.yml      ← downloaded
  ├── docker-compose.yml
  └── ...

        

      


      

        
2

        

          
Build and start all services (takes a few minutes the first time)

          
docker compose up --build -d

# On Linux, if you get "permission denied connecting to Docker":
sudo docker compose up --build -d

# To avoid needing sudo permanently (log out and back in after):
sudo usermod -aG docker $USER

        

      


      

        
3

        

          
Open in your browser

          
http://localhost:8000
